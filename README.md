SpaceMUD 
=============

Text based Multi-User-Dungeon engine created using Unity3d.


Features
-------
* Text based interface with graphical dungeon map 
* Network mutiplayer game (server/client)
* Quick HUD interface
* Create new levels using Tiled Editor
